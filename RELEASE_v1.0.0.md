# GrokChess v1.0.0

**Built by Saad Kamal with xAI's Grok 4.3 (April 2026 release)**

GrokChess is now publicly available as a clean, well-documented open source project.

## What is GrokChess?

A modern, educational chess trainer featuring:

- Three AI difficulty levels powered by Stockfish 16 (WASM) + custom engine
- Real-time coaching that explains moves in **plain English** (no SAN)
- Beautiful holographic-inspired dark UI
- Full client-side computation (no server, no data collection)

## Highlights of v1.0.0

- Clean, production-ready codebase suitable for public scrutiny
- Extracted and well-tested chess logic (`src/lib/chessLogic.ts`)
- Comprehensive unit tests with Vitest (~91% coverage on core logic)
- Professional documentation (README, CONTRIBUTING, SECURITY, CODE_OF_CONDUCT, CHANGELOG)
- Proper MIT licensing and contributor guidelines
- Issue & PR templates for healthy open source collaboration
